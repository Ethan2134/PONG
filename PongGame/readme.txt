PONG

This a game of Pong. Pong is a game with two paddles and a ball that is moving back and forth. The objective is to score
by getting the ball past the paddle of the other player. The left paddle is controlled by using W/S, and the right paddle
is controlled by using the UP/DOWN arrows. The score is displayed at the top of the window, and there is no limit so you
can play as long as you like. The source code is included, as well as an executable JAR file in the 'dist' folder. This
project was created using NetBeans and Java Swing.
